# Carousel

 Carousel Using React + Redux + Webpack 

Libraries:
- React + Redux + Webpack 

**To Run locally:**
```
$ npm install
$ npm run start
```
Try http://localhost:8080

**To Run test:**
```
$ npm run test
$ npm run test:watch
```

**To build for production:**
```
$ npm install
$ npm run build
```
